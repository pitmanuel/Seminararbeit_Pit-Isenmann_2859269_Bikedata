Hier ist eine kurze Erklärung zu den Files:
Folgende Files, sind Modelle, deren Ergebnisse mit Hilfe des Datensatzes "WORKDATA.csv" reproduziert werden kann:
- ARIMA neu
- Benchmark
- GBM
- Lineare Regression
- Prophet
- Random Forest
- XGB

Die Files "enrichment" und "AVG_Stationen hinzufügen" beinhalten den Großteil der Erstellung des Datensatzes "WORKDATA.csv".

"IQR" ist lediglich das File, in dem die Interquartilsabstände ermittelt wurden.
